Turn your Claude subscription into a software factory

Describe what you want to build. otto-factory writes the code, tests it, and reviews it — automatically.

Get started on GitHub

How it works

  1. 1

    Write a PRD

    Describe the feature or product you want to build in plain language — no code required.

  2. 2

    Run the planner

    Run python main.py plan and otto-factory turns your PRD into a prioritised task list automatically.

  3. 3

    Watch it build

    Run python main.py run and sit back while otto-factory writes, tests, and ships the code.

Get started

Clone the repo, add your Claude API key, and start shipping. otto-factory handles the rest.
